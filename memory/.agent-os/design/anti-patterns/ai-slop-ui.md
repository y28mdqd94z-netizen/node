# Generic AI-Slop UI

Problem:
- Layouts and visuals that look templated, interchangeable, or obviously AI-generated.

Why it hurts:
- Loss of credibility and weak product differentiation.

Detection signals:
- User hesitation, low task completion, or inconsistent visual rhythm.

Fix:
- Commit to a clear aesthetic direction and enforce hierarchy through typography and spacing.

Do instead:
- Use design principles and system tokens first.
