# Primitive Composition Recipes

Use these recipes to compose primitives consistently.

## SaaS Admin List Management
- Shell: `list-management-shell`
- Patterns: `crud`, `search-filtering`, `list-detail`
- Components: `table`, `toolbar`, `pagination`, `sheet`

## B2B Dashboard
- Shell: `dashboard-shell`
- Patterns: `dashboard`, `data-heavy`
- Components: `card`, `table`, `toolbar`

## Builder Workflow
- Shell: `builder-shell`
- Patterns: `builder`, `overlays`
- Components: `toolbar`, `sheet`, `tabs`, `input`

## Mobile Utility
- Shell: `mobile-sheet-first-shell`
- Patterns: `overlays`, `form`
- Components: `sheet`, `button`, `input`
