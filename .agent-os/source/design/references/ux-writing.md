# UX Writing (Short-Form)

Rules:
- Labels are verbs when they trigger actions.
- Avoid jargon unless user is expert-only.
- Empty states must guide the next action.
- Error messages must be specific and actionable.

Checks:
- Primary CTA labels are unique and unambiguous.
- Helper text uses sentence case.
