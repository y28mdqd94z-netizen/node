# Color + Contrast (Short-Form)

Rules:
- Body text contrast >= 4.5:1.
- Disabled text contrast >= 3:1.
- Never use color alone to convey status.
- Use 1 accent color for primary actions; avoid rainbow palettes.

Checks:
- Primary CTA passes contrast on both default + hover.
- Focus rings are visible on all surfaces.
