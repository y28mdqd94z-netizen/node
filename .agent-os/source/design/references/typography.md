# Typography (Short-Form)

Purpose: establish hierarchy and tone quickly without generic defaults.

Rules:
- Use 2 type families max (display + body).
- Minimum body size 16px on desktop, 15px on mobile.
- H1-H3 should show clear weight/size steps (no token collisions).
- Avoid long all-caps labels; use sentence case for clarity.

Checks:
- Body line length 55-80 chars on desktop.
- Heading weight contrast is visible at a glance.
