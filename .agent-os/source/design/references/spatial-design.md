# Spatial Design (Short-Form)

Rules:
- Use a clear vertical rhythm (4/8px grid or similar).
- Prefer sections and lists over card walls for primary content.
- Keep primary actions above the fold in key flows.
- Avoid dense clusters; add whitespace before introducing new sections.

Checks:
- One dominant content column per view by default.
- Consistent gutters across breakpoints.
