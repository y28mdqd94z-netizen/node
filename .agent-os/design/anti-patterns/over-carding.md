# Over-Carding

Problem:
- Wrapping every element in cards or nesting cards inside cards.

Why it hurts:
- Visual noise, flattened hierarchy, and poor scanability.

Detection signals:
- User hesitation, low task completion, or inconsistent visual rhythm.

Fix:
- Use structure and spacing; reserve cards for true grouping boundaries.

Do instead:
- Use design principles and system tokens first.
