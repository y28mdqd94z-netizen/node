# Design Principles (Mapped)

Primary principles are defined in:
- `.agent-os/lib/ux-ui-knowledgebase/05_design-system/design-system-principles.yaml`

Active principles to anchor decisions:
- `principle_clarity`: Calm clarity, fast scan, one primary action by default.
- `principle_consistency`: Systems-first consistency, reuse components/tokens.
- `principle_accessibility`: Contrast and keyboard baseline.
- `principle_scalability`: Composable structures, avoid card walls.

Use these as defaults; override only with explicit justification and mitigation.
